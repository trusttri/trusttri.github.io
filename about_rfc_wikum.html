This project is still ongoing, so I will briefly introduce my motivation, contribution, and things I learned so far from this project.


Motivation

Experiencing millions of people protesting against the political corruption in South Korea (2016 - 2017) and witnessing it lead to the impeachment of the previous president was the main motivation.

It was surprising to me that the experts evaluated the social media playing a critical role in assisting the citizens to gather their opinions and organize peaceful candlelight protests. I have always thought it is hard to establish consensus in the SNS, mainly because I experienced it is very hard to resolve disputes regarding controversial matters including sex, gender, and politics.

I wanted to study more about systems that can build consensus on a large scale, further aiding collective action, and asked for voluntary research opportunities in Haystack Group at MIT.


Contribution

My main contributions to this project were the following:

(1) create a novel dataset of Requests for Comment (RfC)
(2) analyze the dataset to earn insights about the RfC process using various techniques including machine learning and correlation
(3) participate in interviewing top RfC participants extracted from the dataset to find issues within the system
(4) design and implement new features in Wikum to resolve existing issues and improve the process


What I found interesting

I found it interesting to:

(1) earn insights about online discussions by utilizing data
(2) design and develop systems to improve collaborative work


Skills I strengthened

Through this project I mostly learned how to:

(1) efficiently create a (large) dataset from the web
(2) freely use various analytical techniques including machine learning
(3) develop systems using web framework and related software (Django, RabbitMQ, Celery) - I fell in love with Django.


Where from here?

I began to have interest in resolving disputes and bridging opposing views online.
More generally, I decided I want to study more about social computing including online discussion and collaboration.
I am also eager to study new areas such as crowdsourcing, since I am aware of the importance of building new systems using human computation.

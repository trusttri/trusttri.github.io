졸업할 때가 가까워져서 그런 것인지, 요새 생각이 너무 많다. 이 글은 새벽 1시쯤에 쓴 글이라, 며칠 후에 너무 부끄러우면 지울 예정!
(English translation: Maybe it's because I'm closer to graduation, I have so many thoughts these days. I'm writing this post around 1AM, so I might delete it a few days later if I feel it's too cringey.)

-쓰고 싶은 말이 너무 많은데, 굳이 하나 뽑자면, "내가 왜 박사 과정을 밟고 있고, 왜 연구를 하고 있지"라는 생각을 저학년 때보다 더 하고 있는 것 같다. 박사 과정을 관둔다는 게 아니라, 연구의 의미에 대해 더 생각하게 되는 것 같다. 어떻게 보면, 연구에 대한 마음가짐이 정말 진지해진 것 같기도 하면서, 또 한편으로는, '5학년이 되기 전에 이 질문에 대한 답을 내야 하는 것 아니었어?!'라는 생각도 든다...
-머리가 너무 복잡하거나 우울할 때에도, 결국에 드는 생각은 "그럼에도 불구하고 지금 쓰는 논문을 마무리해야 한다"라는 거다. 뭐랄까, 이 논문이 세상을 더 나은 곳으로 만들 거라는 바람에 가깝기보다는 (물론 그런 바람은 항상 있지만, 확실히 저학년 때에 비해 지친 것 같다), 조금이라도 논문을 다듬으면서 내 글쓰기 실력과 사고력을 키워야겠다는 생각에 가까운 것 같다. 2년 후, 5년 후, 10년 후, 20년 후에 나는 어디에 있을지 모르겠지만, 그때의 내가 지금의 나보다 글을 더 잘 쓰고 생각 정리를 잘 할 수 있는 사람이라면, 세상에 긍정적인 영향을 좀 더 (조금이라도) 줄 수 있지 않을까,라는 생각으로 자기 위로를 하게 되는 것 같다. 그만큼 글쓰기에 대한 애착이 커졌나보다. 그런데 문제는 한국어로 글을 진짜 잘 못 쓰겠다. 좀 더 일기를 자주 써야겠다는 생각이 든다.
